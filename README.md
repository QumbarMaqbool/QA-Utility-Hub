## ğŸš¦ Getting Started

1. **Prerequisites**

   - Node.js & npm installed
   - Git for version control

2. **Installation**

   ```bash
   # Clone the repository
   git clone https://github.com/QumbarMaqbool/qa-craft-toolkit.git

   # Navigate to project directory
   cd qa-craft-toolkit

   # Install dependencies
   npm install

   # Start development server
   npm run dev
   ```

3. **Build Commands**

   ```bash
   # Production build
   npm run build

   # Development build
   npm run build:dev

   # Preview build
   npm run preview
   ```
